# Teste QA - Cadastro de Usuários

Automatização de teste utilizando framework Cypress


## Informações de versões

- Node.js: 18.17.1
- NPM: 10.1.0
- Yarn: 1.22.19
- Cypress: 13.1.0

### Execução

- Instale as dependências do projeto:

```shell
npm install
```

- Para executar os testes:
```shell
npm run cypress:open
```
